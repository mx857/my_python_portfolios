# Тестирование REST API с pytest и requests

Проект с автоматизированными тестами для проверки базовых операций REST API (GET, POST, PUT, DELETE) на эндпоинте `/posts` публичного API [JSONPlaceholder](https://jsonplaceholder.typicode.com/).

## Описание проекта

Этот проект представляет собой набор автоматизированных тестов, написанных на Python с использованием библиотеки pytest. Тесты проверяют корректность работы CRUD операций для постов блога, а также обработку ошибок и валидацию данных.

### Особенности реализации

- Организация тестов в классах (`TestPosts`)
- Использование фикстур для управления сессиями и тестовыми данными
- Параметризация тестов для проверки множественных сценариев
- Проверка статус-кодов, структуры JSON и содержимого данных
- Негативные тесты для проверки обработки ошибок
- Чистый код с комментариями, соответствующий PEP8
